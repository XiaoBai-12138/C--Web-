# C段扫描工具：XScan-Web

### 初学Python如有瑕疵见谅

### C段扫描TCP协议，可选Web模式支持获取状态码，标题，中间件等

```
### help：python3 XScan.py -h
参数：python3 XScan.py -i 192.168.1.1-5即可
```
